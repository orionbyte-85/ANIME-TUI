# Server Info - Samehadaku TUI

## Server Compatibility

### ✓ Stream-Ready Servers (langsung bisa play di MPV)
Server ini menyediakan direct video URL yang bisa langsung diputar:
- **Pixeldrain** ✓ - Fast, reliable, direct streaming
- **Google Drive** ✓ - Direct streaming via gdriveplayer

### ⚠ Download-Only Servers (perlu download dulu)
Server ini memberikan halaman download, bukan direct video URL:
- **Acefile** - File hosting dengan halaman download
- **Wibufile** - Memerlukan parsing untuk dapat URL video
- **MirrorUpload** - Mirror/redirection service
- **Racaty** - Cloud storage dengan download page
- **Krakenfiles** - File hosting service
- **Gofile** - Premium file hosting
- **Zippyshare** - Memerlukan JavaScript untuk unlock URL
- **Reupload** - Upload/mirror service

## Rekomendasi Penggunaan

1. **Untuk Streaming Langsung**: Pilih server dengan tanda ✓
   - Pixeldrain (prioritas #1)
   - Google Drive

2. **Jika Stream-Ready Tidak Ada**: 
   - Coba server lain dengan yt-dlp support
   - Atau download file terlebih dahulu

## Catatan

Server "Blogspot 360p/480p", "Premium 720p/1080p", dll yang terlihat di website adalah **streaming embeds** yang dimuat via JavaScript. TUI ini fokus pada **download links** yang bisa langsung di-extract dan di-play tanpa browser.

Untuk streaming embed tersebut, Anda perlu menggunakan browser langsung di website Samehadaku.
